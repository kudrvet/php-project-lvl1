## Rules

The BrainGame is cli game. It set consist of five games: 
- brain-calc
- brain-even
- brain-prime
- brain-progression
- brain-gcd

 Answer correctly 3 times in a row - and you a WINNER!.
 
 ## Usage

To start game run it from terminal. 
For example ,

    brain-calc
